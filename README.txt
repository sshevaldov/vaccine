Страница "Перечень вакцинаций"
Путь: "http://localhost/vaccine/admin/admin_cabinet/admin_cabinet.php"
Реализованный функционал:
    1. Автоматическая установка темы страницы.
    2. Возможность ручного изменения темы страницы.
    3. Проверка наличия ввода обязательного поля.
    4. Подгрузка логина пользователя по веденному логину.
    5. Выгрузка перечня вакцинаций в виде PDF или списком.
    6. Установка параметров фильтраций и сортировки искомого перечня, таких как:
        а. Период (сегодня, завтра, конкретная дата, временной промежуток, все время);
        в. Выбор города вакцинаций, либо вывести все города;
        с. Выбор учреждения вакцинации в указанном выше городе;
        d. Выбор сортировки (по умолчанию, город, место вакцинации, дата, паспорт пациента);
    7. Подгрузка учреждений вакцинации по заданному городу.
    7. Ручной переход на страницу авторизации административного интерфейса.
    9. Наличие масок ввода.
