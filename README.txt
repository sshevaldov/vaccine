Страница "Добро пожаловать!"
Путь: "http://localhost/vaccine/user/auth/auth.php"
Реализованный функционал:
    1. Автоматическая установка темы страницы.
    2. Возможность ручного изменения цвета страницы.
    3. Проверка наличия ввода обязательных полей.
    4. Просмотр и скрытие вводимого пароля.
    5. Проверка существования вводимого паспорта в базе.
    6. Проверка правильности пароля.
    7. Ручной переход на страницу регистрации или административного интерфейса.
    8. Автоматический переход в личный кабинет при правильности введенный номера паспорта и пароля.
    9. Наличие масок ввода.

Страница "Личный кабинет"
Путь: "http://localhost/vaccine/user/cabinet/cabinet.php"
Реализованный функционал:
    1. Автоматическая установка темы страницы.
    2. Возможность ручного изменения цвета страницы.
    3. Проверка наличия ввода обязательных полей.
    4. Подгрузка ФИО из базы в соответствии с введенным номером паспорта.
    5. Проверка статуса вакцинации первой прививкой.
    6. Проверка статуса вакцинации второй прививкой.
    7. Изменение списка адресов взависимости от заданного города.
    8. Установка минимальной даты второй вакцинации, если есть первая.
